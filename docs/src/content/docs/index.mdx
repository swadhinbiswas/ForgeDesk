## Build Path

1. Install system prerequisites for Linux, macOS, or Windows.
2. Install the ForgeDesk Python package and optional frontend tooling.
3. Create your first app from the CLI templates.
4. Run with hot reload, then build signed release artifacts.

Follow the full install flow in [Installation](/install/).
