---
title: Docs Platform Migration to Astro Starlight
description: ForgeDesk docs now run on Astro Starlight for Cloudflare Pages deployment.
---

We migrated the ForgeDesk docs portal to Astro Starlight.

## Why this change

- Better long-form docs authoring experience
- Fast static builds for Cloudflare Pages
- Built-in search and clean sidebar information architecture
- Better support for release notes and blog-style content

## What changed

- New docs structure under `src/content/docs`
- Tauri-inspired install journey and onboarding
- Dedicated blog section for releases and updates
