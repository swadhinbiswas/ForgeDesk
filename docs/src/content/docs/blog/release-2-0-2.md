ForgeDesk 2.0.2 focuses on reliability and first-run developer experience.

## Highlights

- Fixed Linux Wayland WebKit compositing crashes for common environments
- Fixed import path issues in the complex app template
- Improved package naming consistency for `forgedesk` and `@forgedesk/*`
- Improved docs consistency and publishing flow

## Upgrade

```bash
pip install --upgrade forgedesk
```
