ForgeDesk installation follows a predictable path similar to modern native-app toolchains:

1. Install OS-level dependencies
2. Install Python and ForgeDesk
3. Optionally install Node.js for frontend frameworks
4. Create and run your first project

## Quick Install

```bash
pip install forgedesk
```
